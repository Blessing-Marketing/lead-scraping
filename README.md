# lead-scraping

## Setup

Erstelle eine `.env`-Datei im Root-Verzeichnis mit folgenden Keys:

```env
close_api_key=DEIN_CLOSE_API_KEY
airtable_api_key=DEIN_AIRTABLE_API_KEY
```

| Key | Beschreibung |
|-----|-------------|
| `close_api_key` | API-Key für [Close CRM](https://www.close.com/) |
| `airtable_api_key` | Personal Access Token für [Airtable](https://airtable.com/) |
